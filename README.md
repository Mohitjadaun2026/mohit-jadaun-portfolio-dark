## ⚙️ How to Run Locally

```bash
# 📁 Clone the repository
git clone <your-repo-url>

# 🚗 Move to the project folder
cd <project-folder>

# 📦 Install dependencies
npm install

# ▶️ Start the development server
npm run dev
```

🌐 Visit: `http://localhost:5173`
